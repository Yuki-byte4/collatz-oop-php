<h1>Collatz OOP Project</h1>

<p>This project implements the 3x+1 (Collatz) problem using Object-Oriented Programming in PHP.</p>

<h2>Features</h2>
<ul>
  <li>Single number calculation</li>
  <li>Interval processing</li>
  <li>Statistics (min/max iterations)</li>
  <li>Histogram (child class)</li>
</ul>

<h2>Technologies</h2>
<p>PHP, OOP, GitHub</p>
